###技术

+   一切都是原生javascript编写，未用任何框架；
+   采用了响应式布局，良好兼容不同宽度大小的屏幕；
+   使用gulp完成前端工程化工作；
+   采用sass对样式进行重写；
+   运用localStorage对数据进行存储，数据保存在自己的浏览器上，所以你可以随意折腾；
+   对输入数据进行转码，进行了XSS防护；
+   对项目分为classify， task， text，overlay，info 5个module进行编写，以及模块加载；
+   移动端翻页效果，用了CSS3 3D transform变换，移动端没有写操作，可以删除文件和编辑原有文件；
+   还有就是项目设计参考了锤子便签
